Возможности

Всё что нужно
для вашего авто

Полный набор инструментов для контроля расходов, обслуживания и статистики

Учёт заправок

Записывайте каждую заправку: объём, цену, тип топлива и АЗС. Полная история всегда под рукой.

Техобслуживание

Планируйте и отслеживайте все виды ТО: замена масла, шин, тормозов и многое другое.

Статистика

Детальные графики расходов по месяцам, расход топлива на 100 км, стоимость километра.

Калькулятор топлива

Рассчитайте стоимость поездки, расход и необходимый объём топлива для любого маршрута.

Цены на топливо

Актуальные цены на топливо на ближайших АЗС прямо в приложении.

Календарь ТО

Визуальный календарь всех плановых обслуживаний. Никогда не пропустите важную дату.

Уведомления

Умные напоминания о предстоящем ТО по пробегу и дате. Всегда в курсе состояния авто.

QR-сканер

Быстрое добавление данных через QR-код. Сканируйте чеки с АЗС мгновенно.

О приложении

Технологии
под капотом

MyCar построен на современном стеке технологий. Android-приложение написано на Kotlin с использованием лучших практик разработки. Серверная часть — FastAPI на Python с облачной базой данных.

Kotlin + Android SDK
FastAPI (Python)
Microsoft SQL Server
WorkManager (фоновые задачи)
Email-верификация (SMTP)
QR-сканер (CameraX)
🔒

Безопасность

Пароли хранятся в виде SHA-256 хешей. Email-верификация при регистрации.

☁️

Облачное хранение

Все данные синхронизируются с сервером. Доступ с любого устройства.

🔔

Умные уведомления

WorkManager отслеживает пробег и даты ТО в фоне, даже когда приложение закрыто.

📊

Аналитика

Расчёт среднего расхода топлива, стоимости километра и месячных трат.
